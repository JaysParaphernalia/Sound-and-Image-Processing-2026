## Directory contents

> modulo_checkerboard_pattern


We were given a code in class to adapt with circles to produce a checkboard grid pattern using modulo.

*Features:*

. Modulo operator for alternating pattern

. Nested Loops

> wk8_WeeklyTask


Our weekly task was to use at least 4 samples to create a drum pattern and make the pattern change in time.

This sketch replicates an essential hip-hop drum pattern the "one two drum". Done via four audio samples (kick, snare, hi-hat and clap).

*Features:*

. Time based sequencing using frameCount

. Rhythm using modulo
